# Hotel Reservation System

A console-based Hotel Reservation System developed using Core Java.

## Features

- View all rooms
- Search rooms by category
- Book rooms
- Cancel reservations
- View reservation details
- Payment simulation
- File handling using bookings.txt

## Technologies Used

- Java
- Maven
- OOP
- Collections (ArrayList)
- File Handling

## Project Structure

```
src
 ├── model
 ├── service
 ├── util
 └── Main.java
```
